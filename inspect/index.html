HTML Inspect esimerkki

1. Avaa inspector

Chrome

Chrome selaimessa voit avata Inspect työkalun painamalla ctrl+shift+c (windows) tai cmd+shift+c (mac) tai hiiren oikealla näytön päällä ja valitsemalla Inspect.

Firefox

Firefox selaimessa voit avata Inspect työkalun painamalla ctrl+shift+i (windows) tai cmd+shift+i tai hiiren oikealla näytön päällä ja valitsemalla Inspect.

Safari

Safarissa sinun tulee ensin ottaa kehittäjätila käyttöön.

  1. Avaa Safari
  2. Valitse Safari > Asetukset
  3. Valitse Lisäasetukset
  4. Valitse Näytä Kehittäjävalikko valikosta
  5. Valitse Kehittäjä > Kehittäjätyökalut

Tämän jälkeen hiiren oikealla näytön päällä ja valitsemalla Inspect Element.

2. Avaa network-välilehti

Network-välilehdeltä näet kaikki sivun lataamat tiedostot. Tiedostot on jaettu eri tyyppisiin ryhmiin. Esimerkiksi HTML, CSS, JavaScript, kuvat, fontit jne.
